Images

Add image / paste / drop Stores the image with the post and inserts the markdown.
![alt](./pic.webp) Colocated image (optimized at build).
![alt](/blog/slug/shot-{theme}.webp) Theme-aware image: renders shot-light.webp or shot-dark.webp to match the visitor's theme. Must live in public/, not colocated.
![A](…webp "Cap A")
![B](…webp "Cap B")		 Two image lines in one paragraph (no blank line between) become a side-by-side comparison row, with the "titles" as captions. Stacks on mobile.
![A](…webp "Cap A")
![B](…webp "Cap B")
[slider]		 Two images followed by a [slider] line become a draggable before/after slider with a 20°-slanted divider (the first image overlays the second). Reacts to hover and touch; an expand button opens both images full size. Write descriptive ![alt] text and a short "title" (the on-image label).
